# Deprecated Automodel Recipes

This directory archives the older NeMo Automodel SFT launchers, training configs,
local Automodel patches, and superseded Qwen3-VL Swift experiments. New Qwen3
SFT experiments should prefer the ms-swift launchers in `../scripts` and the
pure-Qwen3 Swift/FSDP configs in `../configs`.

The spelling `depricated` is intentional to match the requested path.
